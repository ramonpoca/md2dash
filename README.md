# md2dash

A python script to convert a Markdown file into a Dash Cheatsheet

## Setup

```bash
pip install mistune

```

## Usage

Just invoke it with the markdown file:

```bash
./md2dash.py README.md
```

then you can add the docset to dash by double clicking or using `open`

```bash
open README.docset
```
